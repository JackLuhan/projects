美国总统特朗普推特发文称与中国贸易谈判的第一阶段协议接近完成。推文发出，道琼斯指数应声上扬。华尔街股市创新高，凸显市场希望中美尽快达成某种贸易战停火协定的焦急心态。

亚洲股市周五也出现上涨。日经平均指数和香港恒生指数开盘分别上涨2.2和1.9个百分点。中国上证指数早上涨幅超过1%。

在会晤白宫官员后评价说，“这是一个好的开始。”

据悉，白宫官员美国时间周四（12月12日）会晤了一些美国工商界代表，并暗示美中第一阶段贸易协定已经原则上完成，只等待特朗普总统批准，并很可能在周末前会发布相关讯息。

众所周知，如果美中双方无法在白宫制定的12月15日最后期限前达成协定，则美国对中国生产商品的新一轮加征关税措施即将生效。
